# Rancho Típico de Santa Maria da Reguenga

Website institucional desenvolvido para o Rancho Típico de Santa Maria da Reguenga, com o objetivo de representar digitalmente o grupo, a sua história e a cultura tradicional portuguesa.

## Sobre o projeto

Este projeto foi desenvolvido com foco numa identidade visual tradicional, moderna e responsiva, procurando transmitir a essência do folclore português e da cultura minhota.

O website pretende oferecer uma presença digital profissional para uma associação cultural, valorizando:
- tradição
- património cultural
- eventos
- identidade regional

##  Funcionalidades

- Design moderno e responsivo
- Landing page institucional
- Secção sobre o rancho
- Galeria de imagens
- Navegação simples e intuitiva
- Interface otimizada para dispositivos móveis
- Estrutura visual adaptada ao tema cultural

##  Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- React
- Vite

## 🎯 Objetivos do projeto

- Criar uma presença digital moderna para uma associação cultural
- Melhorar competências de frontend development
- Trabalhar UI/UX adaptado a clientes reais
- Desenvolver um projeto institucional para portfólio
